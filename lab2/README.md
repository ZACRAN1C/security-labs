# Лабораторна робота №2: Очищення та аналіз потокових даних

Цей проєкт присвячено попередній обробці (data cleaning), аналізу та візуалізації часових рядів на основі датасету індивідуального енергоспоживання.

## 📊 Про датасет
* **Назва:** Individual Household Electric Power Consumption Dataset
* **Опис:** Містить виміри електричної потужності та суб-метрик споживання з хвилиною дискретизацією.

## 🛠 Вимоги до системи та середовища
* **Інтерпретатор:** Python версії `3.10` або `3.11`
* **Середовище розробки:** Jupyter Notebook (`.ipynb`)
* **Необхідні бібліотеки:** `pandas`, `numpy`, `scipy`, `matplotlib` (або `seaborn`)
