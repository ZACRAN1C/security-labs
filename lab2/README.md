# Лабораторна робота №2: Data Cleaning та аналіз

## Вимоги до системи
- **Python:** 3.10+
- **Середовище:** Jupyter Notebook
- **Бібліотеки:** pandas, scipy, ipykernel

## Інструкція з запуску
1. Активувати venv (за наявності): `.\venv\Scripts\activate`
2. Встановити залежності: `pip install -r requirements.txt`
3. Запустити Jupyter: `jupyter notebook`

## 📂 Робота з даними
- Великі початкові датасети завантажуються в папку `vhi_data/` локально і заігнорені в Git, щоб не перевантажувати репозиторій.
